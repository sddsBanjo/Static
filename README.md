
# Static

 A ideia do nosso projeto é ser um aplicativo para gravar a geolocalização (e futuramente fotos) de momentos especiais dos usuários, para que posteriormente eles possam ver num mapa os pontos onde cultivaram boas memórias e experiências, relembrando com carinho e saudade o passado.
